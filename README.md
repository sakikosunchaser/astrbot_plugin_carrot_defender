# astrbot-plugin-carrot-defender

基于 **AstrBot 4.x** 的 QQ 文字互动小游戏插件。  
玩家可以在 QQ 群聊或私聊中，通过文字指令游玩一个“保卫萝卜”风格的轻量塔防小游戏。

## 新增功能

- 新塔：治疗塔
- 新模式：无尽模式
- 新榜单：无尽排行榜
- 新查询：我的战绩

## 指令列表

- `/萝卜开始`：开始普通模式
- `/萝卜无尽`：开始无尽模式
- `/萝卜状态`：查看当前完整状态
- `/萝卜状态简洁`：查看当前简洁状态
- `/萝卜速览`：查看当前简洁状态
- `/萝卜建造 弓箭 2`
- `/萝卜建造 炮塔 3`
- `/萝卜建造 冰塔 4`
- `/萝卜建造 治疗塔 5`
- `/萝卜升级 2`
- `/萝卜拆除 2`
- `/萝卜下一回合`
- `/萝卜下一波`
- `/萝卜记录`
- `/萝卜排行`
- `/萝卜无尽排行`
- `/萝卜群排行`
- `/萝卜我的战绩`
- `/萝卜结束`
- `/萝卜帮助`

## 治疗塔

- 类型：治疗塔
- 作用：每回合结束时为萝卜回血
- 不会超过最大生命
- 可升级，升级后治疗量提高

## 无尽模式

- 指令：`/萝卜无尽`
- 没有最大波数
- 敌人会随着波数不断成长
- 每 10 波出现强化 Boss
- 排名按无尽最高波数排序

## 排行系统

### `/萝卜排行`
综合排行，显示：
- 胜场
- 普通模式最高波数
- 无尽模式最高波数
- 胜率
- 总击杀

### `/萝卜无尽排行`
显示：
- 无尽最高波数
- 最高生存回合
- 总击杀

### `/萝卜我的战绩`
显示：
- 总局数
- 胜负
- 普通模式最高波数
- 无尽模式最高波数
- 最高生存回合
- 总击杀
- 总赏金
- 总治疗量

## 兼容性

- AstrBot 4.x
- aiocqhttp / OneBot v11

## 持久化

首次运行时会自动创建：

- `data/sessions.json`
- `data/leaderboard.json`

无需手动创建 `data/` 目录。
